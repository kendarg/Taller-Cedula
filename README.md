# Taller-Cedula
## 🪪 Generador de Cédula de Identidad Canina - República de Colombia

Un proyecto web interactivo desarrollado con **HTML5** y **CSS3** que recrea el diseño clásico de la cédula de ciudadanía colombiana, adaptado de forma divertida como una identificación personal para mascotas.

## 🚀 Características

* **Diseño Réplica:** Fiel al formato oficial (Encabezado institucional, número de identificación, apellidos, nombres y sección de firma).
* **Diseño Adaptable (Responsive):** Estructurado con **CSS Grid** y **Flexbox** para garantizar que los datos textuales permanezcan alineados a la izquierda y la fotografía se posicione perfectamente a la derecha sin solaparse.
* **Fondo Moderno:** Implementación de un fondo geométrico dinámico generado mediante SVG en CSS (`background-image`).
* **Estilo Limpio:** Tipografía de sistema moderna, bordes suavizados (`border-radius`) y contrastes definidos para emular un documento de policarbonato real.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructuración semántica del documento y formularios de lectura (`<label>`).
* **CSS3:** Estilizado avanzado, uso de variables de diseño, rejillas (**Grid**) para la composición interna y maquetación flexible.
